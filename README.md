# Discombobulator

Minimal browser image editor for total image discombobulation that exclusively runs on your browser without connection to a backend server.

## Use It

- Online: [https://discombobulator.railway.online](https://discombobulator.railway.online)
- Local download:
  1. Download this repo
  2. Open `index.html` in a browser

Made for certain personal purposes, but feel free to use it.
